Ikuti 3 Langkah Pantas Ini untuk GitHub Pages:

  1. Buka Google Sheets anda di pautan di atas.
  2. Klik Extensions > Apps Script. Padam semua kod lama dan tampal kod di bawah:
function doPost(e) {
  try {
    var sheet = SpreadsheetApp
      .openById("1w-UvCdue0BJvms_o5U0PzU4CoGnp791R")
      .getSheetByName("Book1");

    var data = JSON.parse(e.postData.contents);

    var nextRow = sheet.getLastRow() + 1;

    sheet.getRange(nextRow, 1).setValue(nextRow - 1); // No
    sheet.getRange(nextRow, 2).setValue(data.Nama);
    sheet.getRange(nextRow, 3).setValue(data.CheckIn);
    sheet.getRange(nextRow, 4).setValue(data.CheckOut);
    sheet.getRange(nextRow, 5).setValue(data.NoTel);
    sheet.getRange(nextRow, 6).setValue(data.Email);
    sheet.getRange(nextRow, 7).setValue(data.Alamat);

    return ContentService
      .createTextOutput(
        JSON.stringify({
          status: "success"
        })
      )
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {

    return ContentService
      .createTextOutput(
        JSON.stringify({
          status: "error",
          message: err.toString()
        })
      )
      .setMimeType(ContentService.MimeType.JSON);

  }
}
  1. Klik Deploy > New Deployment. Pilih jenis Web App. Set "Execute as" sebagai Me dan "Who has access" kepada Anyone.
  2. Salin URL Web App yang dijana dan gantikan pembolehubah EXCEL_WEBHOOK_URL di dalam kod JavaScript halaman ini (dalam fail index.html) sebelum melakukan *Push* ke GitHub! Selesai!

Soalan Lazim (FAQ)

Beberapa soalan biasa yang sering ditanya oleh tetamu untuk rujukan pantas anda.

Pukul berapakah waktu daftar masuk dan daftar keluar?

Waktu daftar masuk standard adalah bermula pada jam 3.00 petang dan waktu daftar keluar adalah pada jam 12.00 tengah hari. Waktu daftar masuk awal tertakluk kepada kekosongan bilik.

Berapakah deposit keselamatan yang diperlukan?

Deposit keselamatan sebanyak RM 100.00 diperlukan sebelum atau semasa pendaftaran masuk. Ia akan dikembalikan sepenuhnya dalam masa 24 jam selepas pemeriksaan daftar keluar jika tiada sebarang kerosakan.

Adakah kemudahan memasak disediakan di dapur?

Ya, dapur kami dilengkapi sepenuhnya dengan peti sejuk, ketuhar gelombang mikro, periuk nasi elektrik, dapur gas, cerek elektrik, serta kelengkapan asas hidangan dan memasak yang mesra muslim.
